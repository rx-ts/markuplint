Same as markuplint but extends with sync API.

## Usage

```sh
# yarn
yarn add markuplint-sync

# npm
npm i markuplint-sync
```

### API

```js
import { exec } from 'markuplint-sync'

const result = exec({
  files: './**/*.html',
})

console.log(result) // => The results in JSON format
```
